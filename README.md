# De Patria a Patria

## Descripción del Proyecto

"De Patria a Patria" es un sitio web responsivo que presenta una galería de arte que narra historias de diferentes lugares del mundo. El proyecto muestra un viaje épico desde Kentucky hasta Burundi, pasando por Gales y Ucrania, explorando las ciudades natales de algunos empleados y compañeros de trabajo de Practicum.

## Características

- **Diseño Responsivo**: Adaptado para dispositivos móviles, tablets y escritorio
- **Metodología BEM**: Estructura de CSS organizada y mantenible
- **Flexbox**: Layout moderno y flexible
- **Media Queries**: Breakpoints optimizados para diferentes resoluciones

## Tecnologías Utilizadas

- HTML5
- CSS3
- Metodología BEM
- Flexbox
- Media Queries
- GitHub Pages

## Breakpoints

- **Desktop**: 1280px y superior
- **Tablet**: 768px - 1280px
- **Mobile**: 320px - 768px
